Remembrance Duplication

  • Note: Only Mausoleums with a bell can duplicate all remembrances. Those without a bell specifically cannot duplicate remembrances of Shardbearers.

Remembrances

  • Note: Remembrance items can be obtained only once per playthrough, so duplicate a remembrance only if you want both of its items. There are more Remembrances (25) than duplication coffins (10), so choose wisely.
